# 🎓 Smart Placement & Recruitment Management System (SPaRMS)

A desktop-based Placement & Recruitment Management System developed using **Java Swing**, **JDBC**, and **MySQL**. The application streamlines campus recruitment by enabling students, recruiters, and administrators to manage the complete placement process through role-based dashboards.

---
## 📖 Project Overview

SPaRMS is designed to automate the campus placement process. It replaces manual spreadsheets and paperwork by providing a centralized platform where:

* Students can register, update profiles, manage skills, and apply for placement drives.
* Recruiters can create companies, publish placement drives, review applications, shortlist candidates, and schedule interviews.
* Administrators can monitor placement statistics, manage records, and generate reports.

The application follows a **3-Tier Architecture** consisting of Presentation Layer, Business Layer, and Data Access Layer for better maintainability and scalability.
